📋 API 키 발급 방법
  1. console.cloud.google.com 접속 (구글 계정 필요)
  2. 새 프로젝트 생성 (이름은 아무거나)
  3. 왼쪽 메뉴 → API 및 서비스 → 라이브러리
  4. "YouTube Data API v3" 검색 → 사용 설정
  5. 왼쪽 메뉴 → 사용자 인증 정보 → + 사용자 인증 정보 만들기 → API 키
  6. 생성된 AIza... 키를 위 입력창에 붙여넣기
